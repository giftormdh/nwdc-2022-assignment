Motivasi dan Ekspektasi

Saya memiliki beberapa alasan dan motivasi yang mendorong saya untuk mendaftarkan diri pada NWDC 2022 ini. Yang pertama yaitu untuk menambah pengetahuan, memperluas koneksi dengan orang lain yang memiliki interest yang sama, dan meningkatkan skill saya di dunia pemrograman karena saya masih sangat awam mengenai hal tersebut. Kemudian, dari beberapa pilihan yang ditawarkan, saya tertarik dengan web developer karena cenderung mudah untuk dipelajari dan dikuasai (menurut ekspektasi saya). Alasan yang terakhir yaitu karena saya pernah terpikirkan untuk memiliki website saya sendiri sehingga saya harus bisa menguasai web developer untuk mewujudkannya.

Dari alasan dan motivasi saya tersebut, saya memiliki ekspektasi untuk dapat menguasai web developer dengan maksimal, mampu mengembangkan skill saya secara konsisten bersama rekan-rekan di NWDC, dan dapat memiliki website impian milik saya sendiri dengan adsense yang mengalir lancar sehingga dapat menghasilkan penghasilan tambahan untuk saya.
